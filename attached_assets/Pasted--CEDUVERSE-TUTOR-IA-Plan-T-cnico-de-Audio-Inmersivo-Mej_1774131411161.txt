<a name="a-audio"></a>
## 1. ARQUITECTURA DEL SISTEMA DE AUDIO INMERSIVO

### Visión

Cuando el usuario presiona "Escuchar", debe sentirse como estar en una clase real con un instructor mexicano. No un robot leyendo texto, sino un profesor que:
- Usa ejemplos de la vida real del estudiante
- Hace pausas dramáticas antes de conceptos clave
- Simula interacción con "compañeros" del aula
- Incluye anécdotas, humor y momentos de reflexión
- Referencia normatividad como si la dominara de memoria

### Pipeline completo

```
PASO 1: Claude genera lectureHtml (ya existe)
    ↓
PASO 2: Claude genera classScript (NUEVO — prompt separado)
    ↓ Un guion narrativo conversacional, NO el mismo texto de lectura
    ↓ Incluye marcadores de sección: [INTRO], [CONCEPTO_1], [EJEMPLO], etc.
    ↓ Incluye indicaciones de tono: (enfático), (pausa), (pregunta retórica)
    ↓ ~2,000-3,000 palabras (8-15 min de audio)
    ↓
PASO 3: OpenAI TTS convierte classScript → audio MP3
    ↓ Voz "nova" o "coral" (las más naturales en español)
    ↓ Se cachea el MP3 en filesystem
    ↓
PASO 4: Frontend reproduce con player profesional
    ↓ Play/Pause, seek bar, velocidad, marcadores de sección
    ↓ Auto-scroll sincronizado con el texto de lectura
```

### ¿Por qué un classScript separado del lectureHtml?

El lectureHtml es un documento de referencia extenso (3,000-5,000 palabras) con tablas, listas, normatividad detallada. Está diseñado para LEER.

El classScript es una narrativa conversacional (2,000-3,000 palabras) diseñada para ESCUCHAR. Es como las notas del profesor vs. el libro de texto. Diferente formato, diferente propósito.

Ejemplo de la diferencia:

**lectureHtml (para leer):**
```html
<h2>NOM-035: Obligaciones del Patrón</h2>
<p>La NOM-035-STPS-2018 establece en su numeral 5.1 que el patrón debe:</p>
<ul>
  <li>Establecer por escrito e implementar una política de prevención</li>
  <li>Identificar y analizar los factores de riesgo psicosocial</li>
  <li>Evaluar el entorno organizacional</li>
</ul>
```

**classScript (para escuchar):**
```
Ahora, vamos a hablar de algo que a muchos patrones les da flojera pero que les puede costar MUY caro si no lo hacen. La NOM-035.

(pausa)

Imagínate esto: eres supervisor en una planta de [industria] como la tuya. Un trabajador se queja de que su jefe lo hostiga. Otro se incapacita por "estrés". Un tercero renuncia y dice que el ambiente era tóxico. ¿Qué tienen en común? Todos son factores de riesgo psicosocial. Y la NOM-035 dice que TÚ, como empresa, eres responsable de prevenirlos.

"Pero profe, ¿y qué hago yo como [puesto]?"

Excelente pregunta. Mira, lo primero que la norma te pide es una política de prevención. No es un documento de 50 páginas — es un escrito claro donde la dirección dice: "En esta empresa no toleramos la violencia laboral, y estos son los canales para reportar." Suena simple, ¿verdad? Pues la mayoría de las empresas de [tamaño] como la tuya NO lo tienen...
```

¿Ves la diferencia? El classScript es una clase viva, personalizada, con preguntas retóricas y simulación de aula.

## 2. COMPARATIVA DE APIs TTS

### Los 3 candidatos

| Criterio | OpenAI TTS-1 | OpenAI gpt-4o-mini-tts | ElevenLabs |
|----------|-------------|----------------------|------------|
| **Calidad español MX** | Buena (voces naturales) | Muy buena (más expresiva) | Excelente (la mejor) |
| **Precio** | $15/1M caracteres | ~$0.015/minuto | $5-22/mes (Starter-Creator) |
| **Costo por módulo** (~2,500 chars) | ~$0.037 | ~$0.12-0.15 | Depende del plan |
| **Costo por curso** (4 módulos) | ~$0.15 | ~$0.50 | Dentro del plan mensual |
| **Latencia** | ~0.5s + generación | ~0.5s + generación | 75ms (Flash) a 300ms |
| **Voces español** | Sí (nova, coral, alloy) | Sí (mismas + más control) | Sí (español MX nativo) |
| **Max input** | 4,096 caracteres | 2,000 tokens | 10,000 chars (Multilingual) |
| **Streaming** | Sí | Sí | Sí |
| **SDK Node.js** | openai (oficial) | openai (oficial) | elevenlabs (oficial) |
| **Formato salida** | MP3, WAV, FLAC, PCM | MP3, WAV, FLAC, PCM | MP3, PCM, μ-law |
| **Emocionalidad** | Media | Alta (sigue indicaciones) | Muy alta (natural) |
| **Control de tono** | Limitado | Bueno (via instructions) | Excelente (SSML-like) |

### Mi recomendación: OpenAI TTS-1 para MVP, upgrade a gpt-4o-mini-tts

**Razones:**

1. **Costo predecible:** $0.015 por 1,000 caracteres. Un módulo de 2,500 chars cuesta ~$0.037. Un curso completo ~$0.15. 100 usuarios × 50 cursos = ~$750 total en audio. Manejable.

2. **Misma infraestructura:** Ya usas APIs en el backend. OpenAI se integra con un `npm install openai` y una llamada POST.

3. **Calidad suficiente:** Las voces "nova" y "coral" de OpenAI suenan naturales en español. No son ElevenLabs pero son mucho mejor que Web Speech API.

4. **Sin límites de plan:** Pago por uso, no por suscripción mensual con tope de caracteres.

5. **Upgrade path claro:** Cuando el revenue lo justifique, migrar a `gpt-4o-mini-tts` (más expresivo, sigue instrucciones de tono) o ElevenLabs (la mejor calidad).

### Estimación de costos mensual

```
Supuestos conservadores:
- 500 usuarios activos/mes
- Cada uno genera audio de 2 módulos/mes (promedio)
- Cada módulo: ~2,500 caracteres de classScript
- Total: 500 × 2 × 2,500 = 2,500,000 caracteres/mes

OpenAI TTS-1: 2.5M chars × $15/1M = $37.50/mes
OpenAI gpt-4o-mini-tts: ~$0.015/min × 500 × 2 × 8min = $120/mes
ElevenLabs Starter: $5/mes pero solo 100K chars (insuficiente)
ElevenLabs Scale: $99/mes con 2M chars (ajustado)

Ganador por costo: OpenAI TTS-1 ($37.50/mes para 500 usuarios)
```

<a name="pipeline"></a>
## 3. NUEVO PIPELINE DE GENERACIÓN

### Cambio clave: separar generación de contenido en 2 llamadas

Actualmente una sola llamada a Claude genera todo (lecture + mindmap + quiz + fuentes). Esto causa el bug de JSON truncado porque se piden demasiadas cosas en una respuesta.

**Nuevo pipeline:**

```
LLAMADA 1 a Claude (max_tokens: 6144):
  → Genera: lectureHtml (3,000-5,000 palabras)
  → Genera: mindMap
  → Genera: reflections
  → Genera: suggestedSources
  
LLAMADA 2 a Claude (max_tokens: 4096):
  → Input: el lectureHtml que acaba de generar + perfil del estudiante
  → Genera: adaptiveQuiz (7 preguntas basadas en el contenido real)
  → Genera: classScript (guion de clase de 2,000-3,000 palabras)

LLAMADA 3 a OpenAI TTS (async, no bloquea al usuario):
  → Input: classScript
  → Output: MP3 cacheado en filesystem
  → Se notifica al frontend cuando está listo
```

**Ventajas:**
- Cada llamada tiene un scope más pequeño → menos riesgo de JSON truncado
- El quiz se basa en el contenido REAL generado (no en el esqueleto)
- El classScript se basa en el contenido REAL (coherencia total)
- El audio se genera async — el usuario puede leer mientras se genera
- Si una llamada falla, las otras no se pierden

<a name="json-fix"></a>
## 4. FIX: JSON PARSING ROBUSTO

### Problema actual

```
[ai-engine] Claude API error: Expected ',' or ']' after array element in JSON at position 26062
```

Claude genera JSON largo que a veces se trunca porque `max_tokens: 8192` no alcanza para 5,000 palabras de HTML + mind map + quiz + fuentes todo en un JSON.

### Solución multi-capa

```typescript
// server/ai-engine.ts — Nuevo parseador robusto

function parseClaudeJSON(rawText: string): any {
  // CAPA 1: Limpiar markdown wrapping
  let cleaned = rawText
    .replace(/^```json\s*/i, '')
    .replace(/^```\s*/i, '')
    .replace(/\s*```$/i, '')
    .trim();
  
  // CAPA 2: Intentar parse directo
  try {
    return JSON.parse(cleaned);
  } catch (e1) {
    console.log('[ai-engine] Direct parse failed, attempting repair...');
    
    // CAPA 3: Reparar JSON truncado
    try {
      // Cerrar strings abiertos
      const openQuotes = (cleaned.match(/"/g) || []).length;
      if (openQuotes % 2 !== 0) {
        cleaned += '"';
      }
      
      // Cerrar brackets/braces abiertos
      const opens = { '{': 0, '[': 0 };
      const closes = { '}': '{', ']': '[' };
      
      for (const char of cleaned) {
        if (char in opens) opens[char]++;
        if (char in closes) opens[closes[char]]--;
      }
      
      // Agregar cierres faltantes
      for (let i = 0; i < opens['[' ]; i++) cleaned += ']';
      for (let i = 0; i < opens['{']; i++) cleaned += '}';
      
      return JSON.parse(cleaned);
    } catch (e2) {
      console.log('[ai-engine] Repair parse failed, attempting partial extraction...');
      
      // CAPA 4: Extraer campos individuales con regex
      try {
        const lectureMatch = cleaned.match(/"lectureHtml"\s*:\s*"([\s\S]*?)(?:"\s*,\s*"mindMap|"\s*,\s*"reflections|"\s*})/);
        const mindMapMatch = cleaned.match(/"mindMap"\s*:\s*(\{[\s\S]*?\})\s*,/);
        
        return {
          lectureHtml: lectureMatch ? 
            lectureMatch[1].replace(/\\"/g, '"').replace(/\\n/g, '\n') : 
            null,
          mindMap: mindMapMatch ? JSON.parse(mindMapMatch[1]) : null,
          reflections: [],
          adaptiveQuiz: [],
          suggestedSources: [],
          _partial: true // Flag para indicar que es recuperación parcial
        };
      } catch (e3) {
        console.error('[ai-engine] All parse attempts failed');
        return null;
      }
    }
  }
}
```

### Retry automático

```typescript
async function generateWithRetry(
  prompt: string, 
  maxRetries: number = 2
): Promise<any> {
  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    try {
      const response = await anthropic.messages.create({
        model: "claude-sonnet-4-20250514",
        max_tokens: 6144, // Subir de 8192 para Llamada 1
        // ... resto de config
      });
      
      const text = response.content
        .filter(b => b.type === 'text')
        .map(b => b.text)
        .join('');
      
      const parsed = parseClaudeJSON(text);
      
      if (parsed && !parsed._partial) {
        return parsed; // Parse exitoso completo
      }
      
      if (parsed && parsed._partial && attempt < maxRetries) {
        console.log(`[ai-engine] Partial result on attempt ${attempt + 1}, retrying...`);
        continue; // Reintentar
      }
      
      if (parsed && parsed._partial) {
        return parsed; // Último intento, devolver lo parcial
      }
      
    } catch (error) {
      console.error(`[ai-engine] Attempt ${attempt + 1} failed:`, error.message);
      if (attempt === maxRetries) throw error;
    }
  }
}
```

```
Eres el Tutor IA de Ceduverse. Basándote en el contenido de lectura

[INTRO] — al inicio
[CONCEPTO:nombre] — al iniciar cada concepto principal
[EJEMPLO] — antes de cada ejemplo o caso
[CLAVE] — antes de puntos importantes
[INTERACCION] — simulación de pregunta al aula
[CIERRE] — al final

NO incluir instrucciones de audio como "(pausa)" o "(tono serio)" 
— la voz de TTS no las interpreta. En cambio, usa puntuación natural:

  ],
  "classScript": "El texto completo del guion de clase..."

## 7. NUEVO ENDPOINT DE AUDIO

```typescript
// server/routes.ts — Agregar:

// Generar o recuperar audio de un módulo
GET /api/studio/courses/:slug/modules/:index/audio

// Lógica:
app.get('/api/studio/courses/:slug/modules/:index/audio', async (req, res) => {
  const { slug, index } = req.params;
  const userId = req.user.id;
  
  // 1. Buscar contenido generado
  const generated = await storage.getGeneratedContent(userId, slug, parseInt(index));
  
  if (!generated || !generated.classScript) {
    return res.status(404).json({ error: 'No class script available. Generate content first.' });
  }
  
  // 2. Si ya tiene audio cacheado, servirlo
  if (generated.audioUrl) {
    const audioPath = path.join(AUDIO_DIR, generated.audioUrl);
    if (fs.existsSync(audioPath)) {
      return res.json({ 
        audioUrl: `/audio/${generated.audioUrl}`,
        duration: generated.audioDurationSeconds,
        status: 'ready'
      });
    }
  }
  
  // 3. Si no hay audio, generarlo async
  // Marcar como generando
  await storage.updateGeneratedContent(generated.id, { 
    generationStatus: 'generating_audio' 
  });
  
  // Generar en background (no bloquear la response)
  generateAudioAsync(generated.id, generated.classScript, slug, parseInt(index), userId)
    .catch(err => console.error('[audio] Generation failed:', err));
  
  return res.json({ status: 'generating', estimatedSeconds: 30 });
});

// Polling endpoint para verificar si el audio ya está listo
GET /api/studio/courses/:slug/modules/:index/audio/status

// Servir archivos de audio estáticos
app.use('/audio', express.static(AUDIO_DIR));

### Función de generación de audio

```typescript
// server/audio-generator.ts

import OpenAI from 'openai';
import fs from 'fs';
import path from 'path';

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
const AUDIO_DIR = path.join(process.cwd(), 'audio-cache');

// Crear directorio si no existe
if (!fs.existsSync(AUDIO_DIR)) {
  fs.mkdirSync(AUDIO_DIR, { recursive: true });
}

export async function generateAudioAsync(
  generatedContentId: string,
  classScript: string,
  courseSlug: string,
  moduleIndex: number,
  userId: string
): Promise<void> {
  try {
    const filename = `${courseSlug}_m${moduleIndex}_${userId.slice(0,8)}.mp3`;
    const filepath = path.join(AUDIO_DIR, filename);
    
    // OpenAI TTS tiene límite de 4096 chars por request
    // Si el script es más largo, dividir en chunks y concatenar
    const chunks = splitTextIntoChunks(classScript, 4000);
    const audioBuffers: Buffer[] = [];
    
    for (const chunk of chunks) {
      const response = await openai.audio.speech.create({
        model: "tts-1",
        voice: "nova", // Voz femenina natural, buena en español
        input: chunk,
        speed: 0.95, // Ligeramente más lento para español didáctico
        response_format: "mp3",
      });
      
      const buffer = Buffer.from(await response.arrayBuffer());
      audioBuffers.push(buffer);
    }
    
    // Concatenar buffers (simple append para MP3)
    const finalBuffer = Buffer.concat(audioBuffers);
    fs.writeFileSync(filepath, finalBuffer);
    
    // Estimar duración (MP3 128kbps ≈ 16KB/sec)
    const durationSeconds = Math.round(finalBuffer.length / 16000);
    
    // Actualizar DB
    await storage.updateGeneratedContent(generatedContentId, {
      audioUrl: filename,
      audioDurationSeconds: durationSeconds,
      audioGeneratedAt: new Date(),
      generationStatus: 'complete',
    });
    
    console.log(`[audio] Generated ${filename} (${durationSeconds}s, ${(finalBuffer.length/1024).toFixed(0)}KB)`);
    
  } catch (error) {
    console.error('[audio] Generation error:', error);
    await storage.updateGeneratedContent(generatedContentId, {
      generationStatus: 'failed',
    });
  }
}

function splitTextIntoChunks(text: string, maxChars: number): string[] {
  const chunks: string[] = [];
  const sentences = text.split(/(?<=[.!?])\s+/);
  let current = '';
  
  for (const sentence of sentences) {
    if ((current + ' ' + sentence).length > maxChars) {
      if (current) chunks.push(current.trim());
      current = sentence;
    } else {
      current += (current ? ' ' : '') + sentence;
    }
  }
  if (current) chunks.push(current.trim());
  
  return chunks;
}
```
