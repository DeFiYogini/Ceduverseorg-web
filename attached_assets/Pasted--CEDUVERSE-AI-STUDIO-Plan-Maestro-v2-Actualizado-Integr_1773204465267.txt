## Modelo de Datos PostgreSQL (Nuevas tablas)

```sql
-- ═══════════════════════════════════════════
-- TABLAS NUEVAS PARA STUDIO
-- Se agregan a la base PostgreSQL existente
-- ═══════════════════════════════════════════

-- Cursos del Studio (los 50 cursos estáticos)
CREATE TABLE studio_courses (
  id SERIAL PRIMARY KEY,
  slug TEXT UNIQUE NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  category TEXT NOT NULL,
  subcategory TEXT,
  duration_minutes INTEGER DEFAULT 60,
  level TEXT CHECK (level IN ('basico', 'intermedio', 'avanzado')) DEFAULT 'basico',
  tags TEXT[] DEFAULT '{}',
  
  -- Certificación
  dc3_available BOOLEAN DEFAULT false,
  dc3_price_range TEXT,              -- "$150-$500 MXN"
  nft_certificate BOOLEAN DEFAULT true,
  
  -- Metadata
  icon TEXT,                          -- Emoji o icono
  color TEXT,                         -- Color accent del curso
  source TEXT DEFAULT 'studio',       -- 'studio' | 'procadist' | 'stps'
  
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Módulos de cada curso Studio
CREATE TABLE studio_modules (
  id SERIAL PRIMARY KEY,
  course_id INTEGER REFERENCES studio_courses(id) ON DELETE CASCADE,
  module_index INTEGER NOT NULL,      -- 0, 1, 2, 3
  title TEXT NOT NULL,
  description TEXT,
  content_html TEXT NOT NULL,          -- Contenido base (esqueleto)
  references TEXT[] DEFAULT '{}',
  duration_minutes INTEGER DEFAULT 15,
  
  UNIQUE(course_id, module_index)
);

-- Quizzes del Studio (estáticos, base)
CREATE TABLE studio_quizzes (
  id SERIAL PRIMARY KEY,
  course_id INTEGER REFERENCES studio_courses(id) ON DELETE CASCADE,
  title TEXT NOT NULL,
  passing_score INTEGER DEFAULT 70,
  questions JSONB NOT NULL,           -- Array de {question, options, correctIndex, explanation}
  
  UNIQUE(course_id)
);

-- Perfil extendido del estudiante
-- (se agrega a la tabla de users existente o se crea separada)
CREATE TABLE student_profiles (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id) ON DELETE CASCADE,
  job_title TEXT,
  industry TEXT,
  company_size TEXT CHECK (company_size IN ('1-10', '11-50', '51-100', '101-300', '300+')),
  experience_level TEXT CHECK (experience_level IN ('junior', 'mid', 'senior', 'director')),
  learning_goals TEXT[] DEFAULT '{}',
  preferred_style TEXT CHECK (preferred_style IN ('visual', 'reading', 'audio', 'hands-on')) DEFAULT 'reading',
  
  UNIQUE(user_id)
);

-- Contenido generado por IA (cache)
CREATE TABLE generated_content (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id) ON DELETE CASCADE,
  course_slug TEXT NOT NULL,
  module_index INTEGER NOT NULL,
  
  lecture_html TEXT,                   -- Texto extenso generado
  mind_map JSONB,                     -- JSON del mapa mental
  reflections TEXT[],                 -- Preguntas de reflexión
  adaptive_quiz JSONB,               -- Quiz generado por IA
  suggested_sources JSONB,           -- Fuentes encontradas
  podcast_script TEXT,               -- Script del podcast (Fase 3)
  slides JSONB,                      -- Datos de slides (Fase 3)
  
  personalized_for JSONB,            -- Snapshot del perfil usado
  generated_at TIMESTAMPTZ DEFAULT NOW(),
  
  -- Cachear por usuario + curso + módulo
  UNIQUE(user_id, course_slug, module_index)
);

-- Inscripciones y progreso (unificado Academy + Studio)
CREATE TABLE enrollments (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id) ON DELETE CASCADE,
  
  -- Puede ser curso Academy O Studio
  source TEXT CHECK (source IN ('academy', 'studio')) NOT NULL,
  course_identifier TEXT NOT NULL,    -- academy: course_id | studio: slug
  
  enrolled_at TIMESTAMPTZ DEFAULT NOW(),
  status TEXT DEFAULT 'active' CHECK (status IN ('active', 'completed', 'paused')),
  progress_percent INTEGER DEFAULT 0,
  last_accessed_at TIMESTAMPTZ,
  
  UNIQUE(user_id, source, course_identifier)
);

-- Progreso por módulo/unidad
CREATE TABLE module_progress (
  id SERIAL PRIMARY KEY,
  enrollment_id INTEGER REFERENCES enrollments(id) ON DELETE CASCADE,
  module_identifier TEXT NOT NULL,    -- academy: unit_id | studio: module_index
  completed BOOLEAN DEFAULT false,
  completed_at TIMESTAMPTZ,
  quiz_score INTEGER,
  time_spent_seconds INTEGER DEFAULT 0,
  
  UNIQUE(enrollment_id, module_identifier)
);

-- Certificados emitidos
CREATE TABLE certificates (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id) ON DELETE CASCADE,
  enrollment_id INTEGER REFERENCES enrollments(id),
  
  type TEXT CHECK (type IN ('nft', 'dc3')) NOT NULL,
  course_title TEXT NOT NULL,
  course_slug TEXT NOT NULL,
  
  -- NFT
  nft_token_id TEXT,
  nft_tx_hash TEXT,
  
  -- DC3
  dc3_status TEXT CHECK (dc3_status IN ('pending', 'processing', 'issued', 'delivered')),
  dc3_file_url TEXT,
  dc3_payment_id TEXT,
  dc3_amount INTEGER,                -- Centavos MXN
  
  issued_at TIMESTAMPTZ DEFAULT NOW(),
  quiz_score INTEGER,
  
  UNIQUE(user_id, enrollment_id, type)
);

-- Historial de chat por sesión
CREATE TABLE chat_sessions (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id) ON DELETE CASCADE,
  course_slug TEXT NOT NULL,
  module_index INTEGER NOT NULL,
  messages JSONB DEFAULT '[]',        -- Array de {role, content, timestamp}
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

## Nuevas rutas en server/routes.ts

```typescript
// ═══ STUDIO ROUTES (agregar a routes.ts existente) ═══

// Catálogo de cursos Studio
app.get('/api/studio/courses', async (req, res) => {
  const { category, search, page = 1, limit = 12 } = req.query;
  const courses = await storage.getStudioCourses({ category, search, page, limit });
  res.json(courses);
});

// Detalle de un curso Studio
app.get('/api/studio/courses/:slug', async (req, res) => {
  const course = await storage.getStudioCourse(req.params.slug);
  if (!course) return res.status(404).json({ error: 'Course not found' });
  
  // Incluir módulos y quiz estático
  const modules = await storage.getStudioModules(course.id);
  const quiz = await storage.getStudioQuiz(course.id);
  
  res.json({ ...course, modules, quiz });
});

// Obtener módulo estático (sin IA)
app.get('/api/studio/courses/:slug/modules/:index', async (req, res) => {
  const module = await storage.getStudioModule(req.params.slug, parseInt(req.params.index));
  res.json(module);
});

// ★ GENERAR contenido IA para un módulo
app.post('/api/studio/courses/:slug/modules/:index/generate', async (req, res) => {
  const { slug, index } = req.params;
  const moduleIndex = parseInt(index);
  const userId = req.user.id; // del middleware auth
  
  // 1. Verificar si ya hay contenido cacheado
  const cached = await storage.getGeneratedContent(userId, slug, moduleIndex);
  if (cached) return res.json(cached);
  
  // 2. Obtener datos necesarios
  const course = await storage.getStudioCourse(slug);
  const module = await storage.getStudioModule(slug, moduleIndex);
  const profile = await storage.getStudentProfile(userId);
  
  // 3. Generar con IA
  const generated = await aiEngine.generateModuleContent({
    courseSlug: slug,
    moduleIndex,
    studentProfile: profile,
    courseTitle: course.title,
    moduleData: module
  });
  
  // 4. Cachear en PostgreSQL
  await storage.saveGeneratedContent(userId, slug, moduleIndex, generated);
  
  // 5. Retornar
  res.json(generated);
});

// ★ CHAT contextual con el módulo
app.post('/api/studio/courses/:slug/modules/:index/chat', async (req, res) => {
  const { slug, index } = req.params;
  const { message } = req.body;
  const userId = req.user.id;
  
  // Obtener contexto
  const generated = await storage.getGeneratedContent(userId, slug, parseInt(index));
  const profile = await storage.getStudentProfile(userId);
  const chatSession = await storage.getChatSession(userId, slug, parseInt(index));
  
  // Llamar a IA con contexto
  const reply = await aiEngine.chatWithModule({
    courseSlug: slug,
    moduleIndex: parseInt(index),
    generatedContent: generated?.lecture_html || '',
    studentProfile: profile,
    messages: chatSession?.messages || [],
    newMessage: message
  });
  
  // Guardar mensaje en historial
  await storage.appendChatMessage(userId, slug, parseInt(index), [
    { role: 'user', content: message, timestamp: new Date() },
    { role: 'assistant', content: reply, timestamp: new Date() }
  ]);
  
  res.json({ reply });
});

// Enviar quiz
app.post('/api/studio/courses/:slug/quiz/submit', async (req, res) => {
  const { answers, quizType } = req.body; // quizType: 'static' | 'adaptive'
  const userId = req.user.id;
  
  // Evaluar respuestas contra quiz estático O adaptativo
  const quiz = quizType === 'adaptive' 
    ? (await storage.getGeneratedContent(userId, req.params.slug, 0))?.adaptive_quiz
    : (await storage.getStudioQuiz(req.params.slug))?.questions;
  
  const score = evaluateQuiz(answers, quiz);
  
  // Guardar progreso
  await storage.updateEnrollmentProgress(userId, 'studio', req.params.slug, score);
  
  // Si aprueba (≥70%), marcar como completado
  if (score >= 70) {
    await storage.completeEnrollment(userId, 'studio', req.params.slug, score);
  }
  
  res.json({ score, passed: score >= 70, total: quiz.length });
});

// ═══ PROFILE ROUTES ═══

app.get('/api/profile', async (req, res) => {
  const profile = await storage.getStudentProfile(req.user.id);
  res.json(profile || {});
});

app.put('/api/profile', async (req, res) => {
  const profile = await storage.upsertStudentProfile(req.user.id, req.body);
  res.json(profile);
});

// ═══ CERTIFICATE ROUTES ═══

app.get('/api/certificates/mine', async (req, res) => {
  const certs = await storage.getUserCertificates(req.user.id);
  res.json(certs);
});

app.post('/api/certificates/nft', async (req, res) => {
  const { enrollmentId } = req.body;
  // Verificar que el enrollment está completado
  // Generar NFT (o registrar intención)
  const cert = await storage.issueCertificate(req.user.id, enrollmentId, 'nft');
  res.json(cert);
});

app.post('/api/certificates/dc3/request', async (req, res) => {
  const { enrollmentId } = req.body;
  // Verificar dc3_available del curso
  // Crear solicitud de DC3 pendiente de pago
  const cert = await storage.requestDC3(req.user.id, enrollmentId);
  res.json(cert);
});

// ═══ UNIFIED DASHBOARD ═══

app.get('/api/dashboard', async (req, res) => {
  const userId = req.user.id;
  
  // Combinar datos de Academy + Studio
  const academyEnrollments = await storage.getEnrollments(userId, 'academy');
  const studioEnrollments = await storage.getEnrollments(userId, 'studio');
  const certificates = await storage.getUserCertificates(userId);
  const profile = await storage.getStudentProfile(userId);
  
  res.json({
    profile,
    enrollments: {
      academy: academyEnrollments,
      studio: studioEnrollments,
      total: academyEnrollments.length + studioEnrollments.length
    },
    certificates,
    stats: {
      coursesCompleted: certificates.length,
      totalHoursLearned: calculateTotalHours(academyEnrollments, studioEnrollments),
      currentStreak: calculateStreak(userId)
    }
  });

  │                         │  [Input de mensaje]    │

  │  ▶️ Audio | ⬇️ PDF     │                        │
  ├─────────────────────────┴────────────────────────┤
  │  ✅ NFT Ceduverse al aprobar                     │
  │  📜 DC3 STPS disponible — Desde $150     │
  │  [← Anterior]              [Siguiente →]         │
  └──────────────────────────────────────────────────┘

Flujo:
  1. Al montar: fetch curso estático (/api/studio/courses/:slug)
  2. Verificar si tiene perfil completo → si no, mostrar modal
  3. Al seleccionar módulo: POST /api/studio/courses/:slug/modules/:index/generate
  4. Mostrar skeleton mientras genera (~10-15 seg)
  5. Renderizar contenido en tabs
  6. Chat disponible en panel derecho siempre
  7. Al completar todos los módulos → habilitar quiz final
  8. Al aprobar quiz → emitir NFT + mostrar opción DC3

Estado (React):
  - course: StudioCourse
  - activeModule: number
  - generatedContent: GeneratedContent | null
  - isGenerating: boolean
  - activeTab: 'lecture' | 'mindmap' | 'quiz' | 'sources' | 'podcast'
  - chatMessages: ChatMessage[]
  - quizAnswers: number[]
  - quizSubmitted: boolean
  - quizScore: number | null

Queries (TanStack):
  - useQuery(['studio-course', slug])
  - useMutation(['generate-module'])
  - useMutation(['chat-module'])
  - useMutation(['submit-quiz'])
```

### Diferencias clave

| Aspecto | Academy | Studio |
|---------|---------|--------|
| Fuente de datos | WordPress externo (MCP) | PostgreSQL local |
| Contenido | Curado por instructor | Generado por IA |
| Personalización | No | Sí (puesto, industria) |
| Video | Sí (grabado) | No (audio TTS) |
| Chat IA | No (aún) | Sí |
| Quiz | Auto-generado de títulos | Estático + adaptativo IA |
| Costo infraestructura | WordPress hosting | Claude API tokens |
| Escalabilidad | Limitada por instructor | Ilimitada |
