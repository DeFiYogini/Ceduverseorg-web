# CEDUVERSE — Resumen Técnico y General para Replit

## ¿Qué es Ceduverse?

Ceduverse es una plataforma EdTech (educación + tecnología) que combina inteligencia artificial, certificados blockchain (Web3), y contenido experto para transformar la educación profesional en Latinoamérica. Es el brazo de aprendizaje del ecosistema Ceducap.

**Tagline:** "El universo del aprendizaje inteligente"
**Dominio app:** ceducap.mx
**Dominio academy (WordPress):** ceducap.academy
**Marca visual:** Navy #0D1B4B + Cyan #00C5D4 (brand manual), app usa Blue #1b5adf + Orange #f28023

---

## Visión del Producto

Ceduverse es una plataforma donde:
1. **Estudiantes** se registran, exploran cursos, aprenden con tutor IA, y obtienen certificados verificables en blockchain
2. **Empresas** inscriben equipos, dan seguimiento a progreso grupal, y reciben reportes ejecutivos
3. **Instructores** publican cursos en WordPress y la plataforma los expone automáticamente
4. Los **certificados** se emiten como tokens en Ethereum (Web3) — inmutables y verificables
5. La **gamificación** (logros, rachas, rankings) incentiva la retención

### Stack Principal
- **Framework:** Next.js 16.1.1 (App Router, Server Components, Server Actions)
- **UI:** React 19.2.3 + Tailwind CSS 4 + Lucide icons + Motion (Framer Motion)
- **Auth:** Supabase SSR (@supabase/ssr)
- **Blockchain:** Ethers.js 6 (crear/conectar wallet Ethereum)
- **Validation:** Zod 4
- **Backend de cursos:** WordPress con endpoint MCP custom (JSON-RPC 2.0 sobre HTTP con JWT)
- **Hosting:** TBD (Replit / Vercel / similar)

## API de Cursos (MCP Client)

El `AcademyClient` se comunica con WordPress via JSON-RPC:

```typescript
// Métodos disponibles:
academyClient.getCourses(page, perPage)     // Lista cursos publicados
academyClient.getCourseDetails(courseId)     // Detalle de un curso
academyClient.getCourseCurriculum(courseId)  // Curriculum (secciones + unidades)
academyClient.getUnitDetails(unitId)        // Contenido de una lección
academyClient.getQuizDetails(quizId)        // Preguntas de un quiz
academyClient.getProducts(page, perPage)    // Productos WooCommerce
academyClient.searchCourses(query, page)    // Buscar cursos
academyClient.getStats()                    // Estadísticas del LMS
```

### Tipos principales:
```typescript
interface AcademyPost {
    ID: number;
    post_title: string;
    post_status: string;
    post_date: string;
    post_author: string;
    post_excerpt: string;
    guid: string;
}

interface AcademyCurriculumItem {
    type: 'section' | 'unit' | 'quiz';
    title: string;
    ID?: number;
    unit_type?: string;
    duration?: string;
    index: number;
}
```

## Sobre el Landing Page HTML (V2 Editorial)

El archivo HTML que recibirás es un **landing page estático** diseñado como la cara pública de Ceduverse. Es un single-file HTML con CSS y JavaScript embebido (sin dependencias externas excepto Google Fonts).

### Características del diseño:
- **Estilo:** Light mode editorial, fondo cream #faf8f4, serif headlines (DM Serif Display), body sans-serif (Plus Jakarta Sans)
- **Secciones:** Hero con UI preview cards, features (6), stats bar, cursos (3 cards), proceso (4 pasos), testimonios (3), CTA, footer
- **Animaciones:** Scroll-triggered (Intersection Observer), marquee strip, hover effects
- **Responsive:** Mobile-first con breakpoints
- **Contenido:** En español, orientado al mercado LATAM
